# Code Atlas

A VS Code extension for recording code reading notes.

## Features

- Select code in the editor and pin it as a Markdown file with YAML frontmatter
- Records file path, line range, timestamp, and the selected code
- Outputs to the `code-atlas/` directory in your workspace

## Usage

1. Select code in the editor
2. Press `Ctrl+Shift+P` or open Command Palette (`Cmd+Shift+P`) and run `Code Atlas: Pin Code`
